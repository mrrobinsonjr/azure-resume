# STAR Story Bank – Cloud Architect | Security & Modernization Lead
# STAR 1 — Designing Azure Gov IL5/IL6 Cloud Architecture
(Architecture, Leadership, Mission Delivery)
Situation
USCENTCOM needed to adopt public cloud for mission workloads, but had no existing Azure Government IL5/IL6 environment and strict RMF, SRG, and SCCA requirements.
Task
Serve as the lead Cloud Architect to design and deliver a secure, compliant cloud architecture based on zero trust principals that could support mission systems and achieve ATO.
Action
• Implemented by enforcing micro segmentation and dynamic identity verification
• Built library of terraform modules with RMF controls, this means developers could deploy non compliant resources
• Collaborated with security, networking, and application teams to ensure inheritance and consistency
• Used Terraform and ARM to standardize deployments and reduce configuration drift
•
Result
• Successfully delivered USCENTCOM’s “first” IL5/IL6 cloud environments
• Enabled mission workloads to deploy securely in Azure Gov
• Established reusable architecture patterns adopted by other teams
• Reduced future onboarding time by standardizing the platform
