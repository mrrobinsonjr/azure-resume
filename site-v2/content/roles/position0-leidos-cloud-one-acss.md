## Role Summary

Lead the Azure architecture and engineering practice for the U.S. Air Force Cloud One Architecture and Common Shared Services (ACSS) program, a global-scale, multi-cloud modernization effort supporting mission-critical Department of the Air Force workloads. Combines hands-on cloud architecture with engineering management, coordinating Azure delivery alongside AWS, Google, and Oracle across the program.

## Responsibilities

- Design scalable, resilient, and secure Azure architectures and reference patterns for the Cloud One multi-cloud environment.
- Manage, mentor, and supervise a team of cloud engineers, guiding cloud-native application design, deployment, and complex technical decisions.
- Drive adoption of DevSecOps practices, infrastructure as code, and automation using Terraform, Ansible, Jenkins, and Kubernetes.
- Architect secure cloud solutions implementing zero-trust principles and defense-in-depth across IL4/IL5/IL6 boundaries.
- Manage Azure networking, VPNs, IaaS, PaaS, identity (Azure AD/SAML), and data services supporting shared-services delivery.
- Partner with stakeholders to translate mission requirements into technical solutions and present architectural recommendations to program leadership.

## Impact

Advancing standardized, secure Azure reference architectures and DevSecOps automation across the Cloud One environment to strengthen USAF system resiliency, security, and cost effectiveness.
